What is it for?

xbeam is a set of spreadsheets for engineering calculations in structural analysis, for direct usage onto the web. You find the list of the spreadsheets under Home in the List of Spreadsheets. Check often this list - spreadsheets are updated or new one added regularly!

How does it work?

Spreadsheets are user-friendly and intuitive. The values displayed in blue can be edited. To edit the input value just click it and set a new value.

Who is the author of this project

The author of this project is a structural engineer with 15 years of continuous professionals experiences in the field of structural engineering design of civil and industrial constructions.

How can I save my spreadsheets?

Keep a web link to your spreadsheet if you want to back to it later - the web link include unique identifier dedicated to your spreadsheet. In general spreadsheets are stored 14 days since last time of use.

How to print spreadsheet?

For now the method is to do this with browser's command, preferably by right clicking onto a white region of the spreadsheet and select print option.

Loge python notebook

The project is powered by Python programming language and Loge python notebook. You can create similar spreadsheets using those tools. All those tools are free software. The mission of the Loge project is to provide a simple and practical tool that will interest engineers to use Python language in their daily work.

Contribute

If you enjoy this project you can contribute by reporting problems, shear opinion with others. If you enjoy working with Loge and use it for structural calculations let know if you want shear some of your spreadsheets by xbeam platform. For now the financial contribute is not possible.
